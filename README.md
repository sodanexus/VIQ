# VIQ — Official Website

Official website of electronic music artist **VIQ**.  
Live at: https://www.viqmusic.net

---

## About

This repository contains the source code of the official VIQ website.

It is a minimal, immersive one-page website featuring:

- Full-screen background video
- Animated drawer navigation system
- Custom SVG wave distortion effect
- Smooth entry and exit transitions
- Structured data (Schema.org - MusicGroup)
- SEO optimization (Open Graph, Twitter Cards)
- Fully responsive typography

Built with:
- HTML
- CSS
- Vanilla JavaScript

No frameworks. No dependencies. Pure static website.
